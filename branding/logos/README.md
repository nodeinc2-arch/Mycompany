## Guidance for the animated GIF

- **Concept idea:** the two "nodes" of the wordmark connect (a line draws between them) → then a subtle pulse → loop. Conveys "Node 2 → connection → intelligent infrastructure" without text fatigue.
- **Format:** GIF for email (universally supported); also export WebM/MP4 for web (smaller, smoother).
- **Background:** transparent if possible. Otherwise match brand dark (#0a0a0a) and light variants.
- **File size budget:** ≤2 MB for email-safe.
